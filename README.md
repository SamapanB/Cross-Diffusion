# Cross-Diffusion
Data files and the python files to generate plots related to the article arXiv: 2606.06283 [hep-ph].

There are three types of plots used in the article. The details are listed below:

1. The files with titles of the form "RxyTX036.csv", are to be used in the python file titled "R1_1.ipynb" to generate plots of $\kappa_{xy}/(\tau_{\rm R} T^3)$.

2. The files with titles of the form "RxyXY036.csv", are to be used in the python file titled "R2.ipynb" to generate plots of $(\kappa_{xy} T/\eta)$ as well as $R_T$, whose definition can be found in the article.
