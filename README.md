# Cross-Diffusion
Data files and the python files to generate plots related to the article arXiv: 2606.06283 [hep-ph].

There are three types of plots used in the article. The details are listed below:

1. The files with titles of the form "RxyTZ036.csv", are to be used in the python file titled "R1.ipynb" to generate plots of $\kappa_{xy}/(\tau_{\rm R} T^3)$.

2. The files with titles of the form "RxyZ.csv" and, "RkTnB.csv" are to be used in the python file titled "R23.ipynb" to generate plots of $(\kappa_{xy} T/\eta)$ and, $R_T$ respectively, whose definition can be found in the article.

The data files are generated from Wolfram Mathematica with names R1.nb and, R2n3.nb
